# 一个自己写的脚手架工具
## 安装
```shell
npm install -g honghu-edu-cli
```
## 使用方式
```shell
// 初始化项目
edu-cli init <project-name> (-f)
// -f 强制初始化 , 如果项目存在会覆盖
```

> 然后跟着提示走就是了

| 命令名                 | 作用          |
|---------------------|-------------|
| -h,--help           | 获取帮助文档      |
| init <project name> | 闯进一个初始项目    |
| update              | 更新脚手架       |
| help [command]      | 获取某个命令的帮助信息 |
